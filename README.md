# HPC Lab
Lab exercises on HPC using OpenMP, MPI, SSE/AVX and CUDA

This repo contains the lab exercises for the course *"Software & Programming of High Performance Systems*" of the **Department of Computer Engineering & Informatics** in **University of Patras**.

## Overview
1. **Lab 1:** OpenMP & MPI
   - Hybrid Programming Model using both OpenMP & MPI.
   - Parallel Parametric Search in Machine Learning(Python):
     - Multiprocessing pool
     - MPI futures
     - Master-worker model using MPI
   - OpenMP Tasking.
2. **Lab 2:** Acceleration of WENO5 numerical method and Complex Matrix Multiplication(CMM)
   - Exploration of different SIMD implementations(compiler, OpenMP, AVX) on WENO5.
   - Developement of CUDA kernels and trade-off exploration on CMM.
3. **Lab 3:** Acceleration of Complex Matrix Multiplication
   - OpenMP-based GPU implementation of CMM.
